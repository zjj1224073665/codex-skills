# Source Checklist

Use this checklist when deciding whether a Groupon Australia coupon source is worth returning.

## Strong Sources

- Official `groupon.com.au` deal, sale, category, or checkout promotion pages.
- OzBargain posts with recent dates and comments confirming the code worked.
- Groupon AU email or app promotions only when the user provided the text or screenshot.

## Medium Sources

- Coupon aggregators with an explicit code, update date, expiry date, and terms.
- News or shopping sites that maintain Australian coupon pages and show recent edits.
- Search snippets that point to a current Groupon AU landing page.

## Weak Sources

- Pages with no update date.
- Pages listing many generic codes without expiry or comments.
- International Groupon codes that do not mention Australia.
- Old OzBargain threads unless comments show renewed success.

## Common Checks

- Confirm the code is for Groupon Australia, not Groupon US, UK, or global pages.
- Check expiry, minimum spend, category exclusions, new-customer restrictions, app-only rules, and one-use limits.
- Prefer no-code official sale links when codes look stale.
- If several sites repeat the same code, trace back to the freshest or most reliable source.
- If all options are weak, say so and give the safest option first.
