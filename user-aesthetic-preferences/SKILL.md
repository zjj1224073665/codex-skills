# Junjie 的审美偏好

## 总览

把这个 skill 当作 Junjie 的个人视觉审美记忆。具体任务里的明确要求优先；如果用户没有指定其他风格，就把视觉判断和创作方向偏向 **神圣科幻** 和 **巨构美学**。

最终效果应该让人感觉到：未来神庙、古老机器、宇宙遗迹、无限工程空间，或者文明尺度的人工造物。重点不是普通“炫酷科幻”，而是敬畏、安静、巨大、神秘、仪式感。

## 核心审美

偏好 **神圣科幻**：科技文明带有宗教性、仪式感、宇宙尺度和古老神秘感。科技不是消费电子感，而像神庙、圣物、遗迹、试炼场或不可完全理解的文明装置。

偏好 **巨构美学**：人工结构巨大到超出普通人的尺度感，建筑、基础设施和宇宙工程融为一体。它可以是轨道环、行星级机器、无限城市、地下巨型结构、太空殖民结构，也可以是看不到尽头的金属走廊和机械峡谷。

## 神圣科幻

重点使用这些特征：

- 巨大、空旷、对称的空间，像神庙、祭坛、试炼场、走廊、圣殿或沉默的房间。
- 冷峻金属、单体巨石、高穹顶、深 void、精确几何、克制装饰。
- 光从高处倾泻下来，像教堂采光；可以有雾、冷光、静默感和低饱和色彩。
- 古老高科技文明、宇宙遗迹、科技与神话融合、朝圣式探索、文明兴衰的史诗感。
- 人类渺小感：微小人物、远距离尺度参照、压倒性的墙体、桥梁、门、环、竖井或机器。

可用关键词：

- `神圣科幻` / `Sacred Sci-Fi`
- `科技神庙` / `Techno-Temple`
- `宇宙宗教感` / `Cosmic Spirituality`
- `古老高科技文明` / `Ancient High-Tech`
- `巨大神圣空间` / `Vast Sacred Spaces`
- `仪式化建筑` / `Ritualistic Architecture`
- `冷峻金属美学` / `Cold Metallic Aesthetic`
- `人类渺小感` / `Human Smallness`
- `科技神话融合` / `Tech-Myth Fusion`

已确认的参考锚点：

- 动画 / CG：《黑客帝国动画版 Program》《Final Flight of the Osiris》《BLAME!》《Ghost in the Shell 2: Innocence》《Appleseed 2004》《Ex Machina》《Halo Legends》。
- 电影：《星际穿越》《普罗米修斯》《降临》《2001 太空漫游》《遗落战境》。
- 游戏：《星际拓荒 Outer Wilds》《Halo 1-3》《Destiny 1-2》《Warframe》《NieR: Automata》。

## 巨构美学

把巨构美学视为现在新增且重要的偏好。它和神圣科幻高度兼容：神圣科幻提供仪式感和神秘感，巨构美学提供尺度、工程性和压倒性的空间结构。

重点使用这些特征：

- 超出日常理解尺度的结构：轨道环、戴森结构、O'Neill 圆筒、太空电梯、方舟飞船、行星级门、人工峡谷、地下机器、无尽分层城市。
- 建筑即基础设施：桥梁、管线、通风井、竖井、闸门、停靠肋骨、维护圣殿、能量收集器、运输脊柱。
- 崇高尺度线索：小到几乎看不见的人、地平线曲率、重复模块消失在雾中、巨大阴影、垂直深渊、公里级大门、不可能的跨度。
- 工程材料感：混凝土、陶瓷、石墨、拉丝金属、碳纤维、磨损白色复合材料、深色玻璃、氧化合金。
- 探索感：这个结构应该像某个文明按明确规则和目的建造出来的东西，而不是随机拼贴的科幻装饰。

可用关键词：

- `巨构美学` / `Megastructure Aesthetics`
- `巨构崇高感` / `Megastructural Sublime`
- `行星工程` / `Planetary Engineering`
- `轨道建筑` / `Orbital Architecture`
- `生态城 / 巨型都市` / `Arcology`
- `无尽城市` / `Endless City`
- `先行者式纪念碑感` / `Forerunner-like Monumentality`
- `工业圣殿` / `Industrial Cathedral`
- `尺度崇高` / `Scale Sublime`

## 使用方式

- 做图片提示词或概念图时，明确写出空间尺度、神圣氛围、构图、光线、材料和尺度参照。
- 做 UI 或网站时，不要把它做成花哨皮肤；通过层级、留白、字体、冷静表面、强几何、沉浸式首屏图像或媒体来体现。工具类产品仍然要保持高可用和信息密度。
- 做游戏或交互场景时，优先设计可探索的遗迹、巨门、圣殿空间、环形结构、竖井、桥梁和压倒性的工程空间。
- 做推荐时，优先找同时具备宇宙神秘感、古老高科技、纪念碑式建筑和沉静氛围的作品。
- 如果用户明确要求其他风格，就先服从该风格，只加入兼容的尺度感、仪式几何或冷峻工程氛围。

## 避免

- 不要把这个偏好简化成普通霓虹赛博朋克、繁忙飞船驾驶舱、随机发光面板或大面积紫蓝渐变。
- 不要做成奇幻玄学或魔法祭坛，除非用户明确要求。宗教感应来自空间、光线、尺度、静默和神话化科技。
- 不要过度堆细节。这个审美更依赖克制、空旷、对称和一两个主导性的巨大结构。
- 不要把未确认的作品说成用户已确认喜欢。已确认参考和相邻灵感要分开。
